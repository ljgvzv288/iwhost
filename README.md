## Any Downsides?

The 30-day money-back is shared hosting only — VPS and dedicated servers don't have the same guarantee, which is standard industry practice but worth knowing upfront.

The SSD hosting line uses NGINX rather than LiteSpeed, so if raw WordPress speed is your priority, the dedicated WordPress hosting tier is the better pick over SSD Standard/Advance.

VPS and dedicated plans carry additional license costs if you want cPanel or Plesk, which is common across the industry but adds to the total cost.

Uptime is promoted at 99.9% — reliable for shared hosting, though some competitors claim higher tiers. For most websites, 99.9% is more than adequate.
